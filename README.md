## Featured Project

### DevSetup CLI Tool
A custom DevOps automation CLI tool that sets up a development environment on AWS EC2 with a single command.

Features:
- Installs Git, Docker, AWS CLI, Terraform, and kubectl
- Validates installed tools
- Provides environment cleanup
- Modular CLI commands

Example Usage:

devsetup install
devsetup validate
devsetup clean
